# Library Management System – Project Overview

The Library Management System is a menu-driven application developed using the C programming language to automate basic library operations. The system helps in managing book records efficiently by reducing manual work and improving accuracy.

The primary objective of this project is to provide a simple and user-friendly platform for maintaining library data. The system allows users to add new books, display available books, search for a specific book, issue books to students, return books, and delete book records when required.

The project uses structures, arrays, functions, loops, and conditional statements, which are fundamental concepts of C programming. Each book record contains details such as Book ID, Book Title, Author Name, and Quantity. These records are stored in an array and manipulated through different functions.

This project demonstrates how programming can be used to solve real-world problems in library management. It helps students understand data organization, modular programming, and file handling concepts that can be further extended in advanced versions of the system.

## Objectives

1. To maintain book records efficiently.
2. To automate the process of issuing and returning books.
3. To reduce manual errors in library management.
4. To provide quick access to book information.
5. To apply C programming concepts in a practical application.

## Features

* Add New Book
* Display All Books
* Search Book by ID
* Issue Book
* Return Book
* Delete Book Record
* Menu-Driven User Interface

## Technologies Used

* Programming Language: C
* Compiler: GCC / Turbo C / CodeBlocks
* Concepts Used: Structures, Arrays, Functions, Loops, Conditional Statements

## Expected Outcome

The Library Management System provides an efficient way to manage library records and demonstrates the practical implementation of C programming concepts in a real-world application.
